# Circle Drag and Drop Demo

This project demonstrates a simple web application where circles can be added, selected, and dragged using Angular and ngx-panzoom.

## Getting Started

### Prerequisites

- Node.js and npm

### Installation

1. Clone the repository:
   ```sh
   git clone https://github.com/thibaultfrassati/ngx-panzoom-demo
   cd ngx-panzoom-demo
Install dependencies:
sh
Copy code
npm install
Usage
Start the development server:

sh
Copy code
ng serve
Open your web browser and navigate to http://localhost:4200.

Features
Add circles to the canvas by clicking on it.
Select a circle by clicking on it.
Drag and drop a selected circle to move it.
